class zkynet.models.fcn.FCN(layers)[source]

Bases: zkynet.models.function.Function

FCN: fully-connected networks, also called deep feedforward neural networks, or multilayer perceptrons.

An FCN defines a mapping

y = f(x;p)

where x are the inputs, y are the outputs, and p are the parameters. Specifically,

y = fn (… (f3 (f2 (f1 (x) ) )

where each f(x) = g(Wx + b) is a non-linear function that takes as input a linear transformation.

layer(i)[source]

Returns a layer given index; Note that the first layer (i=0) is closest to the input, while the last layer is the farthest.

class zkynet.models.fcn.FCNLayer(linear_layer, g_func)[source]

An FCN layer is a linear layer passed through a non-linear function.

It outputs: y = g(L(x)) where L(x) is the linear layer

class zkynet.models.function.Function[source]

Bases: object

A function maps inputs to outputs (the forward pass); The function also stores its own gradient functions, for each input.

When the function is called for a forward pass, it also accumulates gradient of the output with respect to every input - that gradient is also a function.
