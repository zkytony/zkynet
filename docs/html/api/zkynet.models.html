Linear layer

class zkynet.models.fcn.Linear(init_weights, init_bias)[source]

Bases: zkynet.framework.computation_graph.Module

A linear layer maps a vector of dimension (n,1) to a vector of dimension (p,1) by:

\(f(x) = x^TW + b\)

where:

  • \(x \in \mathbb{R}^{1 imes n}\)

  • \(W \in \mathbb{R}^{n imes p}\)

  • \(b \in \mathbb{R}^{1 imes p}\)

Here, we accept x in the form of a matrix X of shape (D,n), so the linear mapping can be efficiently computed for many data points.
